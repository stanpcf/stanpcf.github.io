TensorSliceDataset`\n代码使用样例如下:\n\n```python\ndataset1 = tf.data.Dataset.from_tensor_slices(tf.random_uniform([4, 10]))\nprint(dataset1.output_types) # ==> \"tf.float32\"\nprint(dataset1.output_shapes)
